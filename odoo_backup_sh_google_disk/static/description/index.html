Smart rotation

    You can set up along with normal rotation scheme the smart one:
  • Normal: last 10 backups for 10 days
  • Smart: last 7 backups for 7 days and 3 backups for last 3 weeks

Schedule

    You can set up following time frames:
  • In the morning, before office hours, in the afternoon, after office hours
  • Every two hours
  • At night only
  • On sunday only

Backup encryption

    You should enable the encryption option
  • Specify a password to protect data in backup files produced by the backup job
  • Encrypting backups gives you personal control over your personal information

Graph representation

    You can monitor and respond for exceptional following situations in time:
  • backup size has not adequately increased
  • current backup configuration has not failed
  • etc.
